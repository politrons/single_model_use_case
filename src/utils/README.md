# Utility Module

## Overview

This module hosts small helper notebooks and utilities that do not belong elsewhere.

## Notebooks

* **Sleep.py** – Utility notebook that pauses execution for a configurable duration.

## Expected Input Parameters

The notebooks rely on the following *Databricks notebook widgets* (or equivalent job parameters).

| Parameter | Description                                                |
| --------- | ---------------------------------------------------------- |
| `seconds` | Endpoint concurrency setting (e.g., Small, Medium, Large). |
